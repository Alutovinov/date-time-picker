Ng Date Time Picker

A date time picker built by AngularJS

AngularJs DateTimePicker is a cool responsive DateTimePicker directive for Web and Mobile. It is Mobile friendly and light weight.

This package is suitable for any Angular 2 & 4 apps.

  • This Picker is responsive and works on all devices(desktops, tablets and mobiles)
  • This Picker combines both date picker and time picker. You could toggle between them on the show-up picker
  • This Picker has two modes('popup' and 'dropdown'). You could try it in the below demos

Normal pop-up mode:

Click the input, and the date time picker would be pop-up. If you click anywhere out side the pop-up picker, the picker would be close.

Normal inline mode:

In inline mode, the picker would always appear on the page. You could set the [style.width]= "'250px'" to make the picker bigger or smaller. (picker width size between 250px to 300px)

Normal dropdown mode:

Click the text input, and the date time picker would be shown under the text input. If you click anywhere out side the picker, the picker would be close.

Set minMoment and maxMoment:

minMoment and maxMoment would define what valid moment could be selected from the picker. The format is "YYYY-MM-DD HH:mm:ss" (or "YYYY-MM-DD" for only defining min date and max date). In the example, the minMoment is September 7th, 2017 19:30:40('2017-09-07 19:30:40') and the maxMoment is September 17th, 2017 22:50:00 ('2017-09-17 22:50:00').

Choose different theme color. You could provide any 8-Digit Hex color codes. Default Color is '#0070ba'.

Use 12-hour Format:

We can choose different hour format. The default hour format is 24-hour. We could choose the 12-hour format.

Toggle between date picker and time picker:

When the picker open, there is a blue banner on the top, displaying the date and time. When you click the date, the picker would be date picker. And when you click the time, the picker would toggle to time picker.

]]>

Set Different Return Object:

The date-time picker returns a Javascript Date Object by default. Other options could be 'MomentJS Object'(moment), 'Array'(array), 'Json'(json), and 'String'(string).

Set Picker Type:

When the picker open, it will show the calendar (the date picker by default). And when you click the time in the picker banner, it will toggle to the time slider. If you only want to use the date picker or time picker, set the [pickerType] = " 'date' " or [pickerType] = " 'time' "

]]>
]]>

Set Locales:

This date-time-picker has robust support for internationalization. (Default value is 'en'). If you want to change to French for example, just set [locale]=" 'fr' ". You can get more inform about the i18n from MomentJS-i18n.

]]>

Show Seconds Slider:

Show the seconds slider in time picker. Default: false

]]>

Auto Close Picker:

Picker would be automatically close by selecting a day value.
